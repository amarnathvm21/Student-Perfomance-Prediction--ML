# 📘 Student Performance Prediction using Machine Learning

This project predicts student performance using an **XGBoost Regressor** and  
a **Streamlit web application**.

## 📁 Files
- `app.py` → Streamlit application
- `performance_model.pkl` → Trained ML model
- `features.pkl` → Feature Columns
- `scaler.pkl` → Standard Scaler
- `Student_Performance.csv` → Dataset
- `project main.ipynb` → Model training notebook

## 🧠 ML Model
- Algorithm: XGBoost Classifier
- Hyperparameter tuning: GridSearchCV
- Task: Student Performance Prediction 

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
